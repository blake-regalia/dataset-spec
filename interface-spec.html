Data interfaces

Dataset interface

    interface Dataset {
      readonly attribute unsigned long  size;
      Dataset                           add (Quad quad);
      Dataset                           addAll ((Dataset or sequence<Quad>) quads);
      Dataset                           delete (Quad quad);
      Dataset                           deleteMatches (optional Term subject, optional Term predicate, optional Term object, optional Term graph);
      Dataset                           difference (Dataset other);
      boolean                           equals (Dataset other);
      boolean                           every (QuadFilterIteratee iteratee);
      Dataset                           filter (QuadFilterIteratee iteratee);
      void                              forEach (QuadRunIteratee iteratee);
      boolean                           has (Quad quad);
      Promise<Dataset>            import (Stream stream);
      Dataset                           intersection (Dataset other);
      Dataset                           map (QuadMapIteratee iteratee);
      Dataset                           match (optional Term subject, optional Term predicate, optional Term object, optional Term graph);
      Dataset                           merge ((Dataset or sequence<Quad>) quads);
      boolean                           some (QuadFilterIteratee iteratee);
      sequence<Quad>              toArray ();
      String                            toCanonical ();
      Stream                            toStream ();
      String                            toString ();
      Iterator                          iterator ();
    };

Attributes

size A non-negative integer that specifies the number of quads in the set.

Methods

add Adds the specified quad to the dataset. This method returns the dataset instance it was called on. Existing quads, as defined in Quad.equals, will be ignored.

addAll Imports the quads in to this dataset. This method returns the dataset instance it was called on.
This method differes from Dataset.merge in that it adds all quads to the current instance, rather than combining quads and the current instance to create a new instance.

delete Removes the specified quad from the dataset. This method returns the dataset instance it was called on.

deleteMatches This method removes those quads in the current instance which match the given arguments, that is, for each quad in this dataset, it is removed, if:

  • calling quad.subject.equals with the specified subject as an argument returns true, or the subject argument is null, AND
  • calling quad.predicate.equals with the specified predicate as an argument returns true, or the predicate argument is null, AND
  • calling quad.object.equals with the specified object as an argument returns true, or the object argument is null, AND
  • calling quad.graph.equals with the specified graph as an argument returns true, or the graph argument is null

difference Returns a new dataset which contains alls quads from the current dataset, not included in the given dataset.

equals Returns true if the current instance contains the same graph structure as the given dataset. Blank Nodes will be normalized.

every Universal quantification method, tests whether every quad in the dataset passes the test implemented by the provided iteratee. This method will return boolean false when the first quad is found that does not pass the test. Note: this method is aligned with Array.prototype.every() in ECMAScript-262.

filter Creates a new dataset with all the quads which pass the test implemented by the provided iteratee. Note: this method is aligned with Array.prototype.filter() in ECMAScript-262.

forEach Executes the provided iteratee once on each quad in the dataset. Note: this method is aligned with Array.prototype.forEach() in ECMAScript-262.

has Determines whether a dataset includes a certain quad, returning true or false as appropriate.

import Imports all quads from the given stream into the dataset. The stream events end and error are wrapped into a Promise.

intersection Returns a new dataset which contains alls quads from the current dataset, which are also included in the given dataset.

map Returns a new dataset which contains all quads returned by the iteratee, which is called for each quad in the current dataset.

match This method returns a new dataset which is comprised of all those quads in the current instance which match the given arguments as defined in the RDFJS spec Source.match. This method returns a new dataset which is comprised of all those quads in the current instance which match the given arguments, that is, for each quad in this dataset, it is included in the output dataset, if: describe this only once

  • calling quad.subject.equals with the specified subject as an argument returns true, or the subject argument is null, AND
  • calling quad.predicate.equals with the specified predicate as an argument returns true, or the predicate argument is null, AND
  • calling quad.object.equals with the specified object as an argument returns true, or the object argument is null, AND
  • calling quad.graph.equals with the specified graph as an argument returns true, or the graph argument is null
This method implements AND functionality, so only quads matching all of the given non-null arguments will be included in the result. Note, this method always returns a new Dataset, even if that Dataset contains no Quads. Note, Datasets represent Sets of Quads, the order is arbitrary, so this method may result in differing results when called repeatedly.

merge Returns a new Dataset which is a concatenation of this dataset and the quads given as an argument.

some Existential quantification method, tests whether some quads in the dataset passes the test implemented by the provided iteratee. This method will return boolean true when the first quad is found that passes the test. Note: this method is aligned with Array.prototype.some() in ECMAScript-262.

toArray Returns the set of quads within the dataset as a host language native sequence, for example an Array in ECMAScript-262. Note: a sequence in [WEBIDL] is passed by value, not by reference. Note: the order of the quads within the returned sequence is arbitrary, since a Dataset is an unordered set.

toCanonical Returns a N-Triples string representation of the dataset, preprocessed with RDF Dataset Normalization algorithm.

toStream Returns a stream which contains all quads of the dataset.

toString Returns a N-Triples string representation of the dataset. No normalization is done before, so the results for the same quads may vary based on the Dataset implementation.

iterator Returns a ECMAScript 2015 iterator object for the dataset.

DatasetFactory interface

    interface DatasetFactory : DataFactory {
      Dataset dataset (optional (Dataset or sequence<Quad>) quads);
    };

dataset Returns a new dataset and imports all quads, if given.

QuadFilterIteratee interface

    interface QuadFilterIteratee {
      boolean test (Quad quad, Dataset dataset);
    };

test A callable function that returns true if the input quad passes the test this function implements.

QuadMapIteratee interface

    interface QuadMapIteratee {
      Quad map (Quad quad, Dataset dataset);
    };

map A callable function which can be executed on a quad and returns a quad. The returned quad can be the given quad or a new one.

QuadRunIteratee interface

    interface QuadRunIteratee {
      void run (Quad quad, Dataset dataset);
    };

run A callable function which can be executed on a quad.
